# DjangoArticles

Django web app to write articles, save recipes and more...

## Installation
- Use ``git`` command to clone repo
```bash
git clone https://github.com/manemejeff/DjangoArticles.git
```
- Create virtual environment
- Install dependencies from requirements.txt
```bash
pip install -r requirements.txt
```

## Usage
To run dev server, follow these steps
- Change directory
```bash
cd trydjango
```
- Through manage.py file execute runserver command
```bash
python manage.py runserver
```
## Features
- dynamic slugs for article-detail
- automated tests
- pre_save, post_save sginals
- using Q lookups
- custom ModelManager and QuerySet
- admin Inlines
